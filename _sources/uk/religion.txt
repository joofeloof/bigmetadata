.. _uk.ons.LC2107EW0003:

Buddhist Population
---------------------------------------------------------------------------



.. figure:: ../img/uk.ons.LC2107EW0003.png
  :width: 100 %
  :align: center









Measure "Buddhist Population"  density per sq. kilometer  for one point:


.. code-block:: postgresql





  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(51.5146183469, -0.0888347625732),
        'uk.ons.LC2107EW0003'
      );





Measure "Buddhist Population" within an area:

.. code-block:: postgresql

    UPDATE {table_name}
      SET {new_numeric_column} =
        OBS_GetMeasure(
          ST_Buffer(CDB_LatLng(51.5146183469, -0.0888347625732), 0.01),
          'uk.ons.LC2107EW0003'
        );






Measure "Buddhist Population" percent of "Total Population" at one point:

.. code-block:: postgresql

  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(51.5146183469, -0.0888347625732),
        'uk.ons.LC2107EW0003',
        'denominator'
      );


Measure "Buddhist Population" percent of "Total Population" within an area:
