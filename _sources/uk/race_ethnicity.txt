.. _uk.ons.LC2201EW0061:

Asian/Asian British population
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



.. figure:: ../img/uk.ons.LC2201EW0061.png
  :width: 100 %
  :align: center









Measure "Asian/Asian British population"  density per sq. kilometer  for one point:


.. code-block:: postgresql





  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(51.5146183469, -0.0888347625732),
        'uk.ons.LC2201EW0061'
      );





Measure "Asian/Asian British population" within an area:

.. code-block:: postgresql

    UPDATE {table_name}
      SET {new_numeric_column} =
        OBS_GetMeasure(
          ST_Buffer(CDB_LatLng(51.5146183469, -0.0888347625732), 0.01),
          'uk.ons.LC2201EW0061'
        );
