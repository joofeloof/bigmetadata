.. _fr.insee.P12_POP_ETR:

Pop Etrangers en 2012 (princ)
---------------------------------------------------------------------------





Nombre de personnes étrangères





Measure "Pop Etrangers en 2012 (princ)"  density per sq. kilometer  for one point:


.. code-block:: postgresql





  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(48.8608751447, 2.36137390137),
        'fr.insee.P12_POP_ETR'
      );





Measure "Pop Etrangers en 2012 (princ)" within an area:

.. code-block:: postgresql

    UPDATE {table_name}
      SET {new_numeric_column} =
        OBS_GetMeasure(
          ST_Buffer(CDB_LatLng(48.8608751447, 2.36137390137), 0.01),
          'fr.insee.P12_POP_ETR'
        );






Measure "Pop Etrangers en 2012 (princ)" percent of "Population en 2012 (princ)" at one point:

.. code-block:: postgresql

  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(48.8608751447, 2.36137390137),
        'fr.insee.P12_POP_ETR',
        'denominator'
      );


Measure "Pop Etrangers en 2012 (princ)" percent of "Population en 2012 (princ)" within an area:

.. code-block:: postgresql

  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        ST_Buffer(CDB_LatLng(48.8608751447, 2.36137390137), 0.01),
        'fr.insee.P12_POP_ETR',
        'denominator'
      );
