.. _br.geo.setores_censitarios:

Census tracts
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------











Obtain "Census tracts" geometry at one point:


.. code-block:: postgresql





  INSERT INTO {tablename}(the_geom)
    SELECT OBS_GetBoundary(
      CDB_LatLng(-43.19, -22.9),
      'br.geo.setores_censitarios'
    )





Obtain all "Census tracts" geometries within an area:

.. code-block:: postgresql

  INSERT INTO {new_table_name} (the_geom, {geo_id_column})
    SELECT *
    FROM OBS_GetBoundariesByGeometry(
      ST_Buffer(CDB_LatLng(-43.19, -22.9), 0.01),
      'br.geo.setores_censitarios'
    )
