Crimes recorded - Burglary of private residential premises
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------





Crimes recorded - Burglary of private residential premises





Measure "Crimes recorded - Burglary of private residential premises"  density per sq. kilometer  for one point:


.. code-block:: postgresql





  UPDATE {table_name}
    SET {new_numeric_column} =
      OBS_GetMeasure(
        CDB_LatLng(52.5220703614, 13.4060668945),
        'eu.eurostat.crim_gen_regnr_iccs05012'
      );





Measure "Crimes recorded - Burglary of private residential premises" within an area:

.. code-block:: postgresql

    UPDATE {table_name}
      SET {new_numeric_column} =
        OBS_GetMeasure(
          ST_Buffer(CDB_LatLng(52.5220703614, 13.4060668945), 0.01),
          'eu.eurostat.crim_gen_regnr_iccs05012'
        );
