9.1.1. Population in 2010¶

Measure “Population in 2010” density per sq. kilometer for one point:

UPDATE {table_name}
  SET {new_numeric_column} =
    OBS_GetMeasure(
      CDB_LatLng(13.7253777121, 100.492630005),
      'th.thaipop.pop'
    );

Measure “Population in 2010” within an area:

UPDATE {table_name}
  SET {new_numeric_column} =
    OBS_GetMeasure(
      ST_Buffer(CDB_LatLng(13.7253777121, 100.492630005), 0.01),
      'th.thaipop.pop'
    );
